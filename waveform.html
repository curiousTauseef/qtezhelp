Waveform Tab

The Waveform tab allows you to export data to a variety of destinations. If the scope is not stopped, the export operation will stop it as this is the best way to ensure both channels are in sync and that no data changes during export.

The type of exports current supported include:

  • CSV - Comma delimited files you can import to many programs (e.g., Excel)
  • OLS - Open Logic Sniffer data allows that client to do protocol decoding and other analysis on captured data.
  • Sigrok - Sigrok data allows that client to do protocol decoding and other analysis.
  • Plot/Custom - In this mode, the program writes a data file and a script file and runs it through a command of your choosing. Examples are avialable for GNUPLot, for example.

Options

Not all options are available in all modes. The options are:

  • Header - Include a header row
  • Time - Include the time as an output
  • Chan1/2 - Click to include the specified channel in the output. At least one must be checked.
  • Save Config - Save the scope's cuurent configuraiton to the export file.
  • Raw Counts - If checked, the program will export raw counts (0-255) instead of voltages.
  • Logic Threshold - When converting to digital formats (OLS and Sigrok) the program treats anything above this setting as a logic 1 and anything below as a logic 0.

More About Plot/Custom

You can load an example from the bottom left of the popup dialog that appears when you select this type of export. If you make changes, the changes will be saved for next time, but do not alter the example. Practically, this means you can remember one custom setting at any time.

Operation is simple. The program will write a temporarly script file using the contents of the Script box. This script will not be made executable. Any occurance of {FILE} will be replaced by the datafile name. Any occurence of {SCRIPT} will be replaced by the script file name.

The same substitutions will occur on the command before executing it. In some cases, you won't have any script at all. The command might be: 

    cp {FILE} ~/archives
For example. You can also use bash to read the script to write arbitrarily complex actions (see the examples for more on how to do this).

The current examples cover using GNUPLot, Qtiplot, and LibreCalc.
