The Status Tab

The program always starts up showing the status tab. If you haven't used the program before, or you are changing scopes, you may need to enter a different device name in the bottom field. The program will remember the last used device, however, so once you set it up once, you'll probably never need to do it again.

The top part of the tab has several buttons:

  • Connect - Connect to (or disconnect from) the scope.
  • Run/Stop - This is the same as the Run/Stop button on the scope and will appear red or green just like the scope's button does. Note: There is also a red/green toolbar icon that mimics this button.
  • Save USB - This causes the scope to save a screen image to the USB drive.
